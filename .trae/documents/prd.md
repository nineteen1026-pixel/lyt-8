## 1. 产品概述
民宿房间管理系统，面向民宿经营者的纯前端管理工具。
- 主要用途：管理房间档案、查看房间占用日历、登记及维护客人预订信息
- 解决问题：无需复杂后端，直接在浏览器中即可完成民宿日常运营管理
- 目标用户：民宿老板、前台管理员
- 产品价值：轻量、高效、开箱即用的民宿房间管理解决方案

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 管理员 | 本地直接使用（无需注册） | 房间档案CRUD、预订登记/修改/取消、日历查看 |

### 2.2 功能模块
1. **仪表盘首页**：房间概览统计、今日预订摘要、快捷入口
2. **房间档案管理**：房间列表、新增/编辑/删除房间、房间详情
3. **日历视图**：按月日历展示、房间占用/空闲状态、日期快速跳转
4. **预订管理**：预订登记、预订详情查看、预订修改、预订取消

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 仪表盘 | 统计卡片 | 展示房间总数、今日入住、今日退房、空房数 |

## 3. 核心流程
用户进入系统后，可在仪表盘查看概览。主要流程包括：
1. 先在房间管理中录入所有房间信息
2. 通过日历视图查看房间占用情况
3. 点击空闲日期或使用"新增预订"功能登记客人信息
4. 在预订列表中查询、修改或取消预订

## 4. 用户界面设计

### 4.1 设计风格
- 主色：暖棕色 `#8B5A2B`，森绿色 `#5D7B56`
- 辅助色：米白色 `#F5F0E8`，奶油色 `#FFF9F0`
- 强调色：暖橙色 `#D4845E`（已预订），浅灰绿 `#C8D8C5`（空闲）
- 按钮风格：圆角 10px，柔和阴影，hover 微上浮效果
- 字体：标题使用「Playfair Display」展示字体，正文使用「Noto Sans SC」
- 布局风格：左侧导航栏 + 右侧内容区，卡片式内容模块
- 图标风格：Lucide React 线性图标，尺寸统一 18px

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 仪表盘 | 统计卡片 | 渐变背景卡片，大数字 + 副标题，图标点缀，hover 放大动画 |
| 仪表盘 | 今日预订 | 时间轴样式列表，客人信息卡片，入住/退房状态标签 |
| 房间管理 | 房间卡片 | 房间号大字号展示，设施标签，价格高亮，状态色条 |
| 房间管理 | 表单弹窗 | 毛玻璃背景，分组表单字段，柔和阴影 |
| 日历视图 | 日历网格 | 大格子布局，日期号 + 状态圆点，悬停放大效果 |
| 日历视图 | 月份导航 | 简约左右箭头 + 月份标题，回到今日按钮 |
| 预订管理 | 预订列表 | 表格布局，斑马条纹，行悬停高亮 |
| 预订管理 | 操作按钮 | 主色按钮用于提交，次要按钮用于取消，危险色用于删除 |

### 4.3 响应式
- Desktop-first 设计，1280px 为标准宽度
- 平板（768px-1279px）：导航栏保持，内容区自适应
- 移动端（<768px）：导航折叠为汉堡菜单，日历简化显示，卡片堆叠
- 所有触摸元素最小 44px 高度

### 4.4 动画效果
- 页面加载：淡入 + 上移 stagger 动画
- 卡片悬停：y 轴 -2px + 阴影加深
- 弹窗出现：scale 95% → 100% + 淡入
- 日期状态切换：颜色平滑过渡
